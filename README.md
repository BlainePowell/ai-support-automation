# 🎟️ AI-Powered Ticket Handler

Base level ticket support system powered by GPT-4, Zendesk, & Slack

## Setup

1. **Install dependencies**

```
npm install
```

2. **Create .env**

```
ZENDESK_EMAIL=
ZENDESK_API_TOKEN=
OPENAI_API_KEY=
SLACK_URL_GENERAL=
SLACK_URL_BILLING=
SLACK_URL_ENGINEERING_SUPPORT=
```

3. **Start Server**

```
npm run dev
```

4. **Start Ngrok (optional)**

```
ngrok http 3000
```
